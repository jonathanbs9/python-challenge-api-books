## 📖 Simple Book API REST


### 🚀️  Libraries

🇱🇷 Flask==2.0.1
🇱🇷 flask-marshmallow==0.14.0
🇱🇷 Flask-Migrate==3.0.1
🇱🇷 Flask-RESTful==0.3.9
🇱🇷 Flask-SQLAlchemy==2.5.1
🇱🇷 marshmallow==3.12.1
🇱🇷 marshmallow-sqlalchemy==0.26.1
🇱🇷 PyMySQL==1.0.2
🇱🇷 SQLAlchemy==1.4.18


### 🚀️ Build

* *Create virtual environment*

```python -m venv yourVenv```

```pip install <libraries>```


* *Create Mysql database*

```flaskmysql```


### 🚀️ Run

```perl
python src/app.py
```
